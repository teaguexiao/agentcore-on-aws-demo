AgentCore Memory

AgentCore Memory provides persistent memory and context management for AI agents using AWS services.
Persistent Storage

Store agent memory and context in DynamoDB and S3 for long-term persistence.

Vector Search

Semantic search capabilities using Amazon Bedrock Knowledge Bases and OpenSearch.

Session History

Track and retrieve conversation history and agent interactions across sessions.

Context Windows

Intelligent context window management to optimize token usage and relevance.

Memory Compression

Automatic summarization and compression of long-term memory to reduce storage costs.

Shared Memory

Enable memory sharing across multiple agents for collaborative workflows.

Memory Architecture
Short-term Memory
  • In-memory cache using ElastiCache Redis
  • Fast access for recent interactions
  • Automatic expiration and cleanup
Long-term Memory
  • DynamoDB for structured data
  • S3 for large objects and artifacts
  • Vector embeddings in OpenSearch
Memory Management Tools

Memory management interface is coming soon. This will include:
