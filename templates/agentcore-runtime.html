AgentCore Runtime

AgentCore Runtime provides a secure, scalable execution environment for AI agents on AWS infrastructure.
High Performance

Optimized runtime environment with low-latency execution and efficient resource utilization.

Secure Isolation

Each agent runs in an isolated environment with strict security boundaries and resource limits.

Auto Scaling

Automatically scales based on demand using AWS Fargate and Lambda integration.

Session Management

Intelligent session lifecycle management with automatic cleanup and resource optimization.

Network Isolation

VPC-based network isolation with configurable security groups and network policies.

State Persistence

Optional state persistence using S3 and DynamoDB for long-running agent workflows.
